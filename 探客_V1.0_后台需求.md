## 1.用户

##### 1.1 登录

1.1.1 用户手机号登录接口
> 发送：手机号、密码
> 返回：
> - 成功
> ```
>	user_id : "seefhds53346346g4356"
>```
>- 失败
> ```
>	msg : "手机号未注册"或者“用户名或密码有误”,
> ```


1.1.2 用户第三方登录接口
> 是否是第一次第三方登录
> - 是
> 	新创建一个用户，将open_id与新用户绑定，返回user_id
> - 否
> 	返回user_id
>
> 发送：第三方的open_id
> 返回：
> - 成功
> ```
> user_id : "seefhds53346346g4356",
> is_first: true(是否第一次第三方登录)
> ```
> - 失败
> ```
> “第三方登录失败”
> ```

##### 1.2 注册
1.2.1 手机号注册接口

##### 将InstallationId与用户绑定

1. 首页返回游戏列表时，只需要下列信息
	- id （leancloud 自动生成
    - title （游戏的标题
    - cover （link AVFile，游戏的宣传图片
    - promoter (link TUser， 游戏的发起人
    - likes （喜欢游戏的人数 int
    - isTeamGame （0：个人，1：团队 int
    - recommend (0：普通，1：推荐，2：new int
    - state（0：正常，1：内测，2：维护中，3：已结束 int
    - startDate （限定开始日期
    - endDate （限定结束日期
    - startTime （限定开始小时
    - endTime （限定结束小时
    - currentEnrollment （当前报名数
    - limitEnrollment （报名数上限
    - minPPT （每个队伍人数下限
    - maxPPT （每个队伍人数上限
    - latLng （TLatLng 游戏开始坐标
    - city （游戏举办的城市
    - location （举办地点，例如：“洪山区 洪山广场”
